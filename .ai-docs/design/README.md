# Design Folder — Panduan

## Cara Mengisi Folder Ini

1. **Cari inspirasi** di Dribbble, Behance, atau referensi lain
2. **Screenshot** halaman yang menarik
3. **Simpan** ke folder `inspirasi/` dengan nama deskriptif
4. **Update** file `../design.md` dengan deskripsi

## Penamaan File Screenshot

Format: `{halaman}-{sumber}.png`

Contoh:
- `dashboard-dribbble.png`
- `form-input-stitch.png`
- `login-referensi.png`
- `sidebar-inspirasi.png`

## Keyword Pencarian Dribbble

- `admin dashboard clean`
- `document management system`
- `mail management dashboard`
- `form input modern`
- `login page minimal`
- `approval workflow UI`
- `university management system`
- `dark sidebar admin panel`
